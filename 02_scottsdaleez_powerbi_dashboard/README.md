# ScottsdaleEZ Power BI Dashboard

Part of the [AI Business Portfolio](../README.md).

Public portfolio folder for the ScottsdaleEZ Power BI decision-support dashboard.

This section shows how forecast and service-request outputs were translated into operational dashboard views for city staff, with emphasis on forecast interpretation, service performance review, seasonal comparison, and executive communication.

## Included Files

- [Dashboard Refresh Guide](DASHBOARD_REFRESH_GUIDE.md) — public-facing workflow summary for refreshing and validating a Power BI dashboard sourced from monthly prediction-pipeline CSV outputs.
- [Public Poster](scottsdaleez_public_poster_contacts_removed.pdf) — City-approved public poster with contact details removed.

## Dashboard Purpose

The dashboard was designed to help users understand where next-month service request demand may be highest, how service activity changes over time, and how forecast outputs should be interpreted for operational review.

## Public Portfolio Note

This folder includes public-facing summary materials only. It does not include raw data, full CSV outputs, Power BI source files, cloud paths, credentials, private operational materials, or restricted source files.
