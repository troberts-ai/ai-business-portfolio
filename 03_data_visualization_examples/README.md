# Selected Tableau Visualization Work

Public portfolio folder for selected Tableau data visualization examples.

This section includes Tableau workbook files, representative screenshots, short project summaries, and design notes from projects focused on dashboard design, visual storytelling, chart selection, and decision-oriented business communication.

The goal is to show how data can be translated into clear, usable business insight for nontechnical audiences.

## Contents

- Tableau workbook files
- Representative dashboard screenshots
- Short project summaries
- Design notes explaining chart and layout choices

## Public Portfolio Note

These materials are curated for public professional review. Files and screenshots are limited to polished examples and should not include private data, restricted course materials, or sensitive information.
